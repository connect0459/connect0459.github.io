## Resources
### composer
以下のパッケージをcomposerディレクトリ以下にインストール。
```
{
  "require": {
    "phpmailer/phpmailer": "^6.8",
    "google/apiclient": "2.12.1",
    "michelf/php-markdown": "^2.0"
  }
}
```
### css
* reset.cssには[destyle.css](https://github.com/nicolas-cusan/destyle.css)(v4.0.0)を使用。
